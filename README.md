# connected_minds
Connected Minds is a specialized web platform for remote learning, created to support uninterrupted education during the ongoing war in Ukraine.

Keatures include:
- Homework assignment and submission
- Real-time chat
- Electronic gradebook to track academic performance
- Role-based access for students, teachers, and school administrators
- Creating and maintaining learning schedule
- Managing subjects
- Managing students

Built with Next.js, MySQL, and focused on usability.
